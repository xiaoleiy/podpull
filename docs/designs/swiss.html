Section 01

How it works

Apple show / id ──( iTunes Lookup )──▶ RSS feed url RSS feed ──( <enclosure> )───▶ direct .mp3 / .m4a 小宇宙 link ──( og:audio )──────▶ direct .m4a download ──( resumable )─────▶ YYYY-MM-DD - title.ext
Section 02

Three moves

01

Search

Find any show by name.

$ podpull search "睡前故事"
02

Pick

Interactive multi-select list.

$ podpull get 1532755821
03

Download

Resumable, with progress.

$ podpull get --latest 3
Section 03

Specification

A

Interactive picker

Arrow-key, multi-select checklist.

B

Live progress

Size · speed · ETA bars.

C

Apple · RSS · 小宇宙

Shows, feeds, or a pasted link.

D

Resumable

Range requests resume drops.

E

Scriptable

Path on stdout, UI on stderr.

BYOK summariessoon
