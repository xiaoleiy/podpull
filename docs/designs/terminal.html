how it works

apple show / id ──iTunes lookup──▶ rss feed url rss feed ──<enclosure>───▶ direct .mp3 / .m4a 小宇宙 link ──og:audio──────▶ direct .m4a download ──resumable─────▶ YYYY-MM-DD - title.ext

three moves

01

search

find any show by name — iTunes' open catalog.

$ podpull search "睡前故事"
02

pick

give a show, choose from an interactive checklist.

$ podpull get 1532755821
03

download

resumable, with a live progress bar.

$ podpull get --latest 3

features

  • Interactive picker — arrow keys, multi-select
  • Live progress — size · speed · ETA
  • Apple · RSS · 小宇宙 — or a pasted link
  • Resumable — range requests
  • Scriptable — path on stdout
  • BYOK summariessoon
